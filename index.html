Bot Features

  • /appeal - Appeal bans
  • /report - Report users
  • /banlog - Set up a ban log channel
  • /botstats - Get bot statistics
  • /find - Search the ban database
  • /invite - Get invite links
  • /toggle - Enable/disable ban updates
  • /updatebans - Process old bans
  • /help - Get bot commands and info
